2023 ICPC Seoul Regional Apricot Seeds 题解

更新于2024-11-29|题解
|评论数:

题意

给你一个长度为 nnn 的序列，有 mmm 次询问，每次有五个参数 [l r k x y][l~r~k~x~y][l r k x y]。

表示区间 [l,r][l,r][l,r] 单独拿出来，做 kkk 次冒泡排序后，区间 [x,y][x,y][x,y] 的和。

做法

参考 Zaunese。

pjudgepjudgepjudge 提供的做法是枚举一个阈值然后转成 0/1 序列，个人觉得不好理解。

我们模拟两组数据：

发现了 xk+1 x1 x2 … xk  (x1,x2,…,xk≤xk+1x_{k+1}~x_1~x_2~\dots~x_k~~(x_1, x_2, \dots , x_k \leq x_{k+1}xk+1​ x1​ x2​ … xk​  (x1​,x2​,…,xk​≤xk+1​)

会变成 x1 x2 … xk xk+1x_1~x_2~\dots~x_k~x_{k+1}x1​ x2​ … xk​ xk+1​ 。

这启示我们可以尝试从前缀寻找关系（对于询问的区间 [x,y][x,y][x,y] ，可以视为 [1,y]−[1,x−1][1,y]-[1,x-1][1,y]−[1,x−1]）。

对于一次操作，我们尝试考虑前缀 [1,i][1,i][1,i] 进行一次 冒泡操作 的变成的串。

Hint 1

如果一段进行了冒泡的极长子串 xk+1 x1 x2 … xkx_{k+1}~x_1~x_2~\dots~x_kxk+1​ x1​ x2​ … xk​ 所对应的 aj aj+1 … aj+ka_j~a_{j+1}~\dots~a_{j+k}aj​ aj+1​ … aj+k​ 满足 j+k≤ij+k\leq ij+k≤i ，那么这段前缀的 sumsumsum 不会被影响。

证明：所有冒泡影响的操作均在 [1,i][1,i][1,i] 中进行，当然没有影响。

Hint 2

如果一段进行了冒泡的极长子串 xk+1 x1 x2 … xkx_{k+1}~x_1~x_2~\dots~x_kxk+1​ x1​ x2​ … xk​ 所对应的 aj aj+1 … aj+ka_j~a_{j+1}~\dots~a_{j+k}aj​ aj+1​ … aj+k​ 满足 j+k>ij+k> ij+k>i ，那么这段前缀的 sumsumsum 会变成区间 [1,i+1][1,i+1][1,i+1] 的前 iii 小的值的和。

证明：我们对 j+k=i+1j+k=i+1j+k=i+1 与 j+k>i+1j+k>i+1j+k>i+1 分别进行证明。

首先经过一次冒泡排序后前缀 [1,i][1,i][1,i] 会变为 a1 a2 … aj+1 aj+2 … aj+k aja_1 ~ a_2 ~ \dots ~ a_{j+1} ~ a_{j+2} ~ \dots ~ a_{j+k} ~ a_{j}a1​ a2​ … aj+1​ aj+2​ … aj+k​ aj​

（注意此时的 aka_kak​ 表示的是值，不是下标为 kkk 的元素。此时的定义与下面不同）。

因为 aj+k<aja_{j+k}<a_jaj+k​<aj​（注意此时的 aka_kak​ 表示的是下标为 kkk 的元素，不是值），所以前缀 [1,i][1,i][1,i] 的 sumsumsum 是区间 [1,i+1][1,i+1][1,i+1] 的前 iii 小的值的和。


此时有 max{a1,a2,…,aj+k}>ai+1max\{ a_1,a_2,\dots,a_{j+k} \} > a_{i+1}max{a1​,a2​,…,aj+k​}>ai+1​, aj=max{a1,a2,…,aj+k}a_{j}=max\{ a_1,a_2,\dots,a_{j+k} \}aj​=max{a1​,a2​,…,aj+k​}。

反证即可，如下：

  1. 若 max{a1,a2,…,aj+k}≤ai+1max\{ a_1,a_2,\dots,a_{j+k} \} \leq a_{i+1}max{a1​,a2​,…,aj+k​}≤ai+1​ ，则与 j+k>ij+k>ij+k>i 矛盾，故 max{a1,a2,…,aj+k}>ai+1max\{ a_1,a_2,\dots,a_{j+k} \} > a_{i+1}max{a1​,a2​,…,aj+k​}>ai+1​。

  2. 若 aj≠max{a1,a2,…,aj+k}a_{j}\not =max\{ a_1,a_2,\dots,a_{j+k} \}aj​​=max{a1​,a2​,…,aj+k​} ，则值为 max{a1,a2,…,aj+k}max\{ a_1,a_2,\dots,a_{j+k} \}max{a1​,a2​,…,aj+k​} 的，在区间 [1,i][1,i][1,i] 最右边的一个元素会一直冒泡移动至 i+1i+1i+1。

    与此时 xk+1 x1 x2 … xkx_{k+1}~x_1~x_2~\dots~x_kxk+1​ x1​ x2​ … xk​ 是极长子串矛盾。

综上，前缀 [1,i][1,i][1,i] 的 sumsumsum 会变成区间 [1,i+1][1,i+1][1,i+1] 的前 iii 小的值的和。

我们尝试把 Hint 2 推广至 lll (注意字母) 次冒泡排序后的结果。

结论：前缀 [1,i][1,i][1,i] 的 sumsumsum 在经历 lll 次冒泡操作后，会变成 [1,i+l][1,i+l][1,i+l] 的前 iii 小的值的和。

考虑对前缀 [i,1][i,1][i,1] 进行归纳。

  1. 当 l=1l=1l=1 时，上所述以证明。

  2. 当 l=l′+1l=l'+1l=l′+1 时，我们对于一段进行了冒泡的极长子串 xk+1 x1 x2 … xkx_{k+1}~x_1~x_2~\dots~x_kxk+1​ x1​ x2​ … xk​ 分类讨论：

    为了方便，设 xk+1 x1 x2 … xkx_{k+1}~x_1~x_2~\dots~x_kxk+1​ x1​ x2​ … xk​ 在原序列中对应 aj aj+1 … aj+ka_j~a_{j+1}~\dots~a_{j+k}aj​ aj+1​ … aj+k​(不保证 j+k≤ij+k\leq ij+k≤i) 。

    • 当 j+k≤ij+k\leq ij+k≤i 时，所有操作均在前缀 [1,i][1,i][1,i] 内进行，对区间 [1,i][1,i][1,i] 的 sumsumsum 不会产生影响。

      特殊的，当 j+k=ij+k=ij+k=i 时，因为这段子串是极长的，所以有 aj+k≤ai+1a_{j+k}\leq a_{i+1}aj+k​≤ai+1​ ，仍不会对前缀 [1,i][1,i][1,i] 的 sumsumsum 不会产生影响。

    • 否则有 j+k>ij+k>ij+k>i 。此时会把 max{a1,a2,…,aj+k+l−1}max\{ a_1,a_2,\dots,a_{j+k+l-1} \}max{a1​,a2​,…,aj+k+l−1​} 从前缀 [1,i][1,i][1,i] 的 sumsumsum 中减去，并加入 aj+k+la_{j+k+l}aj+k+l​。

    此时前缀 [1,i][1,i][1,i] 的 sumsumsum 变为前缀 [1,i+l][1,i+l][1,i+l] 的前 iii 小的值的和。

综上所述，前缀 [1,i][1,i][1,i] 的 sumsumsum 在经历 lll (注意字母) 次冒泡操作后，会变成 [1,i+l][1,i+l][1,i+l] 的前 iii 小的值的和。

即 ret=ret−max{a1,a2,…,ai,ai+1}+ai+1ret=ret-max\{ a_1,a_2,\dots,a_{i},a_{i+1} \}+a_{i+1}ret=ret−max{a1​,a2​,…,ai​,ai+1​}+ai+1​
